# 100 Great Humans

This repository collects concise, original summaries of the 100 people ranked in Michael H. Hart's book *The 100: A Ranking of the Most Influential Persons in History* (1992).

The source list and one-line role labels are taken from the History Wiki entry on the book, while all prose in this repository is newly written and intended as a neutral, high-level overview of each person’s historical impact.

See `100-great-humans.md` for the full list and descriptions.
